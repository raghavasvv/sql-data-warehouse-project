# sql-data-warehouse-project
Building DWH with SQL SERVER and includes ETL process, data modelling and analytics

**Project Requirements:**

**Building the Data Warehouse:**
**Objective :** Develop modern data warehouse using SQL server to consolidate sales data, enabling analytical reporting.

#### Specifications
**Data Sources:** Import Data from 2 source systems(CRM and ERP) provided as CSV files.
**Data Quality:** Cleanse and resolve data quality issues.
**Integrations:** Combine both sources into single, user friendly data model
**Scope:** Focus on latest dataset, historical data is not required.
**Documentation:** Provide clear documentation of data model to support business stakeholders and internal teams.
